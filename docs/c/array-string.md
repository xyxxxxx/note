# 数组

定义数组m[i]有元素m[0],m[1],...,m[i-1]

```c
int m1[5]={1,2,3,4,5};
int m2[ ]={1,2,3,4,5,6};
int m3[5]={0};
int m4; /*未赋值的元素会成为随机数*/

char ch[6]="CHINA"; /*会补足一位/0*/
char ch[6]="CH"; /*后4位会补足/0*/

int a[2][3]={{1,2,3},{4,5,6}};
int a[2][3]={1,2,3,4,5,6};
int a[][3]={1,2,3,4,5,6}; /*只可以省略行下标*/
int a[2][3]={{1,2},{4}};/*会补足0*/

a[i]+j==&a[i][j];/*a[i]+j表示i行j列的地址 */
*(a[i]+j)==a[i][j];/* *()表示取地址的内容 */

    
```



数组名可作为指针或数组传入
