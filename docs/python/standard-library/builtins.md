# builtins——内建对象

`builtins` 模块提供对 Python 的所有内置对象的直接访问，例如 `builtins.open` 是内置函数 `open()` 的全名。

大多数应用程序通常不会显式访问此模块，但在扩展内置对象时会很有用。
